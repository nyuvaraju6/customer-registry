# User Flow

## Customer Registry Workflow

1. User opens the application.
2. User registers or logs in.
3. Dashboard is displayed.
4. User manages customer records.
5. User registers complaints.
6. Complaint status is updated.
7. Data is stored in MongoDB.
8. Reports are generated.
9. User logs out.
