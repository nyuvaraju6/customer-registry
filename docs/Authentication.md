# Authentication

## Authentication Method

JWT (JSON Web Token)

---

## Features

- User Registration
- Secure Login
- Password Encryption using bcrypt
- Protected API Routes
- Session Authentication
- Token Verification
- Logout

---

## Workflow

User Registration

↓

Password Encryption

↓

Store User

↓

Login

↓

Generate JWT Token

↓

Access Protected Routes

↓

Logout
